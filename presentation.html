The Dataset

The dataset used for develop the app contains 100 measurements of the speed of light from five experiments, each of this consisting of 20 consecutive runs.

The speed of light is coded in km/sec with 299000 subtracted.

The data selection

The user is able to exclude the more extreme values using two limits values: the inferior quantile and the superior quantile, this is achieved obtaining from the user the desired quantile and calculating the corresponding speed value. Its shown in the graph as blue lines:

Plots

After selecting the data, the user can generate the following additional plots:

  • Boxplot: a boxplot segregated by experiments
  • Histogram: a histogram which can be modified by changing their bins
  • Density: a probability density plot using a gaussian kernel, the user can select to see the overall behavior or by experiment.
